APT Repository for Filecoin Software

A signed Debian APT Software Repository for Filecoin software.

Current packages include curio-cuda and curio-opencl from Curio Storage, Inc.

Installation

To install the Filecoin APT repository and the available packages, follow these steps:

  1. Add the repository to your system's sources list:
    2. echo "deb http://filecoin-project.github.io/apt stable main" | sudo tee /etc/apt/sources.list.d/curio-repo.list
  2. Import the repository's GPG key:
    3. wget -O - http://filecoin-project.github.io/apt/KEY.gpg | sudo apt-key add -
  3. Update the package list:
    4. sudo apt update
  4. Install the desired package:
    5. sudo apt install curio-cuda

    or

    sudo apt install curio-opencl
